# day-sources-cli

CLI + GitHub Pages demo to look up day-related information from ranked sources.

## Scope
This MVP implements source selection for **1** and **3**:
- **1** = UN International Days
- **3** = WHO / UNESCO / EU institution days

## Install
```bash
npm install
```

## Run
```bash
node src/cli.js --date 2026-02-28 --sources 1,3
node src/cli.js --date 2026-02-28 --sources 1,3 --json
```

## Test
```bash
npm test
```

## GitHub Pages
The website in `docs/` loads `docs/data-latest.json` generated by:
```bash
npm run build:data
```

You can regenerate and push that file anytime.
